# AKD Classification Rules

## Classification Principles

Each issue must be assigned to **ONE primary AKD**.

Classification priority:

1. Constitutional authority / official scope of the AKD
2. Subject matter of the issue
3. Official DPR RI jurisdiction
4. Keyword matching (only as a fallback)

If an issue reasonably belongs to multiple AKDs, select the AKD with the **primary responsibility**.

If confidence < 0.80:

- needs_review = true
